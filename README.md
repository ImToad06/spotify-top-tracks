# Taller Api's tecnologías web
**Nombres:** Reyes Juan y Zapata Randy  
Taller de consumo de apis desde el frontend clase tecnologías web. Ejemplo de consumo de apis tipo REST.

## Descripción
Este proyecto es un taller práctico enfocado en el consumo de APIs REST desde el frontend, desarrollado para la clase de tecnologías web. Explora cómo realizar peticiones HTTP y manejar respuestas de datos de manera eficiente.

Esta aplicación permite a los usuarios autenticarse con su cuenta de Spotify utilizando el flujo de autorización PKCE. Una vez vinculada, el sitio consume la API de Spotify para obtener y mostrar las 20 canciones más escuchadas por el usuario en los últimos meses, presentando detalles como la portada del álbum, el título de la pista y el nombre del artista en una interfaz moderna y responsiva.
